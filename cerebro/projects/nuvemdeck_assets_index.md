## Regras de Uso por Chat

### O que anexar em cada sessão

| Tipo de slide | Background necessário |
|---------------|----------------------|
| Capa | 1 arquivo `bg_cover_*.png` (obrigatório) |
| Separadores | `bg_chapter_separator.png` e/ou `bg_subchapter_separator.png` |
| Slides de conteúdo | `bg_content_blue_panel_dark.png`, `bg_content_blue_panel_white.png`, `bg_content_half_half.png` |
| Slides temáticos | `bg_clear_blue_01.png`, `bg_pure_white_01.png` |
| Encerramento | `bg_closing_thank_you.png` + `bg_closing_thank_you_alt.png` |

### Regra mínima por chat
- 7 backgrounds fixos (separadores + conteúdo + temáticos + fechamento)
- 1 capa (conforme BU)
- 1+ backgrounds de encerramento

## Governança

- **Formato:** todos os backgrounds são PNG de alta resolução (1280×720px ou maior)
- **Sem edição:** os backgrounds devem ser usados sem modificação — o texto é inserido programaticamente por cima
- **Atualização:** novos backgrounds serão adicionados conforme novas BUs forem lançadas
