# Governance

This repository is the source of truth for the RevOps team's knowledge base ("Cérebro RevOps").

Every `.md` file under `cerebro/` and `onboarding/` is automatically synced to Confluence (space **Ops & Analytics → Cerebro RevOps**) on every push to `main`.

For full governance guidelines — what to document, how to structure files, and how to keep content up to date — see [`onboarding/governanca_cerebro.md`](onboarding/governanca_cerebro.md).
